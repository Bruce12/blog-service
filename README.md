# 博客后台系统管理接口

## 技术

egg + mongodb + typescript + graphql

## 已完成接口列表

- `/users/token`校验用户信息接口，登录需要调此接口获取授权
- `/user/authenticate`刷新token，需要拿token进行请求刷新
- `/v2/dict`文章类型接口，可获取所有文字类型
- `/v2/essay` 文章， restfull，修改和编辑需要单独增加授权

**数据库创建时间问题解决**每个表默认都需要这个字段
